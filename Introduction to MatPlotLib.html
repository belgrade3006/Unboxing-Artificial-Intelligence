Introduction to MatPlotLib:¶

  • Matplotlib is used for data visualization in Python.
  • It’s an open-source library used to generate static, animated, and interactive visualizations.
  • All histograms, line graphs and scatter plots in python are made using matplotlib
  • In order to use matplotlib, we first have to import it

# Creating a circle 
t = np.linspace(0, 2*np.pi, 100)
# The command above divides the interval (0,2*Pi) into 100 equal regions and stores them inn t, a numpy array
circle = np.array([np.cos(t), np.sin(t)]) # Parametric equation of a circle : x=cos t, y=sin t ; t=parameter
# np.cos(t) instantaneously evaluates the cosine of all the elements in the t array, same for np.sin(t)
plt.plot(circle[0], circle[1])
plt.axis('equal') # This line ensures that the both axes are scaled equally, so that the circle doesn't look like an oval
plt.grid(True) # inserts grids in the plot
plt.axhline(0, color='black', lw=1) #Boldens the x axis : axhline -> Axis Horizontal line
plt.axvline(0, color='black', lw=1) #Boldens the y axis : axvline -> Axis Vertical line

#Creating a histogram

# 1. Generate 1,000 random "exam scores" centered around 75
# (Using a normal/Gaussian distribution)
np.random.seed(42)  # Keeps the data identical every time you run it
scores = np.random.normal(loc=75, scale=10, size=1000)

# 2. Create the histogram
# 'bins=20' splits the data range into 20 bars
# 'edgecolor' adds a border between the bars so they don't blend together
plt.hist(scores, bins=20, color="skyblue", edgecolor="black")

# 3. labels and titles
plt.title("Distribution of Exam Scores")
plt.xlabel("Exam Score Ranges")
plt.ylabel("Number of Students (Frequency)")

# 4. Add a grid on the y-axis to make frequencies easier to read
plt.grid(axis="y", alpha=0.75, linestyle="--")

# 5. Display the plot
plt.show()

# Creating a scatter plot

# 1. Scatter plot data
temperature = np.array([65, 68, 70, 75, 80, 82, 85, 88, 90, 95, 100])
sales = np.array([150, 180, 210, 300, 320, 410, 400, 490, 530, 600, 680])

# 2. Calculate the trendline math
# np.polyfit(x, y, 1) means we want a 1st-degree polynomial (a straight line)
slope, intercept = np.polyfit(temperature, sales, 1)

# Calculate the predicted Y values using the line equation (y = mx + b)
best_fit_y = slope * temperature + intercept

# 3. Create the scatter plot
plt.scatter(
    temperature,
    sales,
    color="coral",
    s=80,
    edgecolor="black",
    label="Actual Sales",
)

# 4. Overlay the best-fit line
# We use standard plt.plot() for this!
plt.plot(
    temperature,
    best_fit_y,
    color="darkblue",
    linestyle="--",
    linewidth=2,
    label="Best-Fit Line",
)

# 5. Labels, Legend, and Show
plt.title("Ice Cream Sales vs. Temperature (with Trendline)", fontsize=14)
plt.xlabel("Temperature (°F)", fontsize=12)
plt.ylabel("Sales ($)", fontsize=12)
plt.grid(True, linestyle="--", alpha=0.5)
plt.legend()  # Vital to show which line is which!

plt.show()
